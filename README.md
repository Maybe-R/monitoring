5. Опишите основные плюсы и минусы pull и push систем мониторинга.

5.1. Pull система:
      Плюсы:
            - простота настройки
            - возможность опрашивать только необходимые
            - встроенный health check
            - легче производить отладку системы
      Минусы:
            - нагрузка на сеть
            - сложнее настраивать сетевой доступ
            

5.2. Push система
      Плюсы:
            - обновление данных в реальном времени
            - возможность опрашивать только необходимые
            - масштабируемость
            - подходит для короткоживущих задач
      Минусы:
            - все настраивается через метрики (если агенты умрет, то метрика будет потеряна)
            - сложнее настраивать конфигурацию, при изменениях адреса приложения необходимо деплоить все заново
            - сложнее производиь отладку системы
